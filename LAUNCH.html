2
Create the database
Run from the project root — seeds 2,041 companies + 6,942 executives automatically. 
# Option A — seed from SQL file (fastest)
psql $DATABASE_URL -f seed_data.sql

# Option B — let the server auto-seed on first boot
pnpm --filter @workspace/db push   # create tables
Click to copy
3
Start the application
One command runs both the API and serves the frontend on a single port. 
# Production (built bundle — fastest)
node artifacts/api-server/dist/index.cjs

# Development (hot-reload, TypeScript direct)
pnpm --filter @workspace/api-server dev
# In a second terminal:
pnpm --filter @workspace/prospect-sa dev
Click to copy
4
Open the app
Once the server starts, this page will detect it automatically and load the app inline above. Or navigate directly: http://localhost:3000

Python Scout microservice (optional — needed for CR stealth scraping): 
cd artifacts/python-scout
uv run uvicorn main:app --port 8099
